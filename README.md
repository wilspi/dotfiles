# dotfiles

### Clone and run `run_setup.sh` to setup dotfiles:  
```
git clone https://github.com/wilspi/dotfiles
cd dotfiles
./run_setup.sh
```


### Copy and edit personal settings:  
```
cp .secrets.example ~/.secrets
vim ~/.secrets
```
